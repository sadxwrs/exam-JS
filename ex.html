      РАЗБОР КОДА
    

    

      
let x = 0;
console.log(x ? "yes" : "no");

      
Что выведет код выше?

      
Ответ: "no"

      
Объяснение: 0 — это falsy-значение в JavaScript. Поэтому условие считается ложным и выполняется вторая часть тернарного оператора.

    

  


  

    

      
Ключевые выводы

      
    
        
  • Условия проверяются сверху вниз
    • 
        
  • Выполняется только первый истинный блок
    • 
        
  • Тернарный оператор: условие ? true : false
    • 
        
  • falsy значения: 0, "", null, undefined, NaN, false
